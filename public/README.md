# Static Profile Headshot Asset

To display your professional headshot inside the portfolio's Hero section:

1. Locate your professional portrait (the headshot you uploaded in the chat).
2. Save it as `profile.jpg` (or convert it to a JPEG format).
3. Drag and drop the file directly into this `/public` folder inside your code editor (or root folder).
4. Refresh the preview! The portfolio will automatically load your headshot and apply premium grayscale-to-color editorial hover effects.

*An elegant typographic monogram fallback is pre-configured and will automatically display if the image file is missing or has a different name.*
