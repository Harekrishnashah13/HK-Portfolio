## Key Features

**Structured Case Studies**
Each project includes business context, technical approach, trade-off reasoning, lessons learned, and toolchain breakdown. Recruiters can scan a card in 10 seconds or read the full write-up in 5 minutes.

**Credential Verification**
Every certification shows a real credential ID with a link to the official verification registry.

**Recruiter-First Contact UX**
LinkedIn and Email as primary actions. Structured contact form for detailed enquiries. 4-business-hour response SLA.

**ATS-Ready Resume**
One-click PDF download formatted for applicant tracking systems.

**Performance**
WebGL background disabled on mobile. Lazy-loaded images. Inter font preloaded to prevent FOUT.

## Running Locally

```bash
# Clone
git clone https://github.com/Harekrishnashah13/harekrishnashah-portfolio.git
cd harekrishnashah-portfolio

# Install
npm install

# Start dev server
npm run dev

# Build
npm run build
```

No environment variables required.

## Deployment

Auto-deployed to Vercel on push to `main`.

```bash
npm i -g vercel && vercel --prod
```
